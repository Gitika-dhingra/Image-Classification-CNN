# 🌿 Plant Disease Classification using Deep Learning

## 📌 Overview

This project focuses on detecting plant diseases from leaf images using deep learning models. It compares a custom CNN with transfer learning models (MobileNetV2 and EfficientNetB0) and improves reliability by analyzing both overall and class-wise performance. Explainable AI techniques are also used to make model predictions more transparent.

---

## 🚀 Key Highlights

* Deep learning-based plant disease detection
* Comparison of CNN, MobileNetV2, and EfficientNetB0
* Focus on class-wise performance (not just accuracy)
* Integration of XAI methods: Grad-CAM, LIME, SHAP

---

## 📊 Results

EfficientNetB0 achieved the best performance (~94.5% accuracy) with more balanced results across classes.
